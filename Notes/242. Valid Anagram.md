# 242. Valid Anagram

### Approach 1 — Sort Both Strings

Sort both strings and compare.

- anagrams produce identical sorted order
- Time: `O(n log n)`
- Space: depends on implementation

---

### Approach 2 — Frequency Counter (Optimal)

Count character frequencies.

- increment using first string
- decrement using second string
- all counts must end at zero

Using a fixed-size array for lowercase English letters:

```go
[26]int
```

- Time: `O(n)`
- Space: `O(1)`

General Unicode solution:

```go
map[rune]int
```

- Time: `O(n)`
- Space: `O(k)` where `k` = unique chars
