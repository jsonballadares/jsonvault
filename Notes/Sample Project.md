# Sample Project

## Goal

Demonstrate the shape of a project note. Project notes track ongoing work — goals, tasks, and a log of decisions and progress — with enough structure to be useful without being ceremonial.

## Tasks

- [ ] First task in the project
- [ ] Second task, possibly linking to a ticket note
- [x] A completed task, left checked as a reference

## Log

- **YYYY-MM-DD** — Initial sketch of the project. Open questions about scope.
- **YYYY-MM-DD** — Decision made: narrow scope to the first milestone. See [[Sample Permanent Note]] for context.
